Research

Neurons are relatively simple computation units. However, the brain, a collection of neurons, possesses highly intricate memory functions. For example, episodic memory allows humans to rapidly encode information upon a single exposure and retrieve it later to support adaptive behavior – to do so, the brain constantly needs to decide when to encode a new memory or when to retrieve an old one. What's the strategy the brain uses to coordinate episodic memory in conjunction with other memory systems? Does the optimal strategy explain how human use memory in the real world? How can we implement a system with neuron-like processing units to achieve human-like memory functions?

To investigate computational principles of learning and memory, I use artificial neural networks as model organisms. Then, I test model predictions with behavioral and neuroimaging experiments. By recapitulating human behavior in a neural network model, we can gain insights into the cognitive architecture the brain uses and the statistical structure of the environment we live in.

Understanding the computational/neural/cognitive mechanisms of human memory would make timely contributions to i) effective algorithms for NeuroAI, ii) better solutions to memory-related issues (e.g., brain fog, aging, Alzheimer's disease) iii) informed principles of learning and education.

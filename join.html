I have RA, PhD, and postdoc positions available!
Come work with me on neural network models + behavioral/neuroimging experiments on human memory and NeuroAI!
CityUHK is an English-speaking environment and a very international university, so we welcome students from all over the world!
Contact: qlu.neuro@gmail.com

Research Assistant

  • Ideal qualifications:
    • Bachelor's degree in Computer Science, Neuroscience, Psychology, or related fields
    • Interest in computational modeling and human memory research
    • Experience with human neuroimaging/behavioral experiments or machine learning is a plus
    • Programming skills (Python preferred)

Ph.D. Student

  • Ideal qualifications:
    • Bachelor's or Master's degree in Computer Science, Neuroscience, Psychology, or related fields
    • Interest in computational modeling and human memory research
    • Research experience in human neuroimaging / behavioral experiments or machine learning / computational modeling
    • Strong programming skills (Python preferred)

    • I admit students via the Neuroscience PhD program. The CityU PhD program info page has a lot of information, including application deadlines, additional funding/scholarship opportunities. Please read the info pages carefully and make sure you are fully aware of the "Entrance Requirements" of the program. The start date is flexible, if the regular September start date is too early/late!

Postdoctoral Researcher

  • Ideal qualifications:
    • Ph.D. in Computer Science, Neuroscience, Psychology, or related fields
    • Relevant publication record
    • Expertise in human neuroimaging/behavioral experiments or machine learning / computational modeling

How to Apply

To apply, please send the following materials to qlu.neuro@gmail.com:
  • CV/Resume
    • please indicate your GPA (e.g. 3.5/4.0, 85/100, etc.) -- this is related with the CityUHK's entrance requirement
    • please highlight any relevant research experience you have
  • A brief description of what would you like to work on with me!
